filterM

filterM :: forall a m. (Monad m) => (a -> m Boolean) -> [a] -> m [a]

Filter where the predicate returns a monadic Boolean. For example:

powerSet :: forall a. [a] -> [[a]]
powerSet = filterM (const [true, false])

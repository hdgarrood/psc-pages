A lazy linked list type.

This type is strict in its head element, but lazy in its tail.

Various operations on lazy lists require evaluation of the entire list, so care is needed when defining and using infinite lists.

toArray :: forall a. (List a) -> [a]

Convert a lazy list into an immutable array. This function will attempt to evaluate the entire list, so should only be used on finite inputs.

Running time: O(n) where n is the number of elements in the list.

fromArray :: forall a. [a] -> List a

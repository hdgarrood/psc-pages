Type class for enumerations. This should not be considered a part of a numeric hierarchy, ala Haskell. Rather, this is a type class for small, ordered sum types with statically-determined cardinality and the ability to easily compute successor and predecessor elements. e.g. DayOfWeek, etc.

Laws:

  • succ firstEnum >>= succ >>= succ ... succ [cardinality - 1 times] == lastEnum
  • pred lastEnum >>= pred >>= pred ... pred [cardinality - 1 times] == firstEnum
  • e1 `compare` e2 == fromEnum e1 `compare` fromEnum e2
  • forall a > firstEnum: pred a >>= succ == Just a
  • forall a < lastEnum: succ a >>= pred == Just a
  • pred >=> succ >=> pred = pred
  • succ >=> pred >=> succ = succ
  • toEnum (fromEnum a) = Just a
  • forall a > firstEnum: fromEnum <$> pred a = Just (fromEnum a - 1)
  • forall a < lastEnum: fromEnum <$> succ a = Just (fromEnum a + 1)
